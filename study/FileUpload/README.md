FileUpload
==========

SWFUpload를 이용한 다중파일 업로드 for JAVA

[이클립스를 이용한 실행방법]
- File>Import>Mave>Existing Maven Projects
- File>Maven>Maven Update
- And then http://localhost:8080/FileUpload Access!!
